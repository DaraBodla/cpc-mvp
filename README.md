# Vercel Backend API

A full-featured serverless backend API built with TypeScript for Vercel deployment.

## Features

- ✅ TypeScript with strict mode
- ✅ JWT Authentication
- ✅ Role-based access control (user/admin/moderator)
- ✅ Password hashing with bcrypt
- ✅ Rate limiting
- ✅ CORS handling
- ✅ Request validation
- ✅ Pagination support
- ✅ Error handling middleware
- ✅ Request logging
- ✅ Users CRUD
- ✅ Posts CRUD with drafts/publish workflow
- ✅ Search and filtering
